This is the default progress bar with default width ,color, min and max .

Can be Added in any div anywhere in the page.

The progress bar is set up using user inputs for min , max, current value, width and color.

The button is used to increase the progress in the progress bar by given parameter. The parameter is currently given as 5. Try clicking the button many times.

Test for invalid current value.

Simple example to show how the progress bar can be used in simple form filling application.
